# Development Setup

## 1. 环境要求

- Node.js 20
- npm
- Git

建议使用 `.nvmrc` 中指定的 Node 版本。

---

## 2. 获取项目

```bash
git clone <your-repo-url>
cd <your-repo-name>
git checkout feature/integral-lab-v1
```

## 3. 安装依赖

```bash
npm install
```

## 4. 开发与构建

```bash
# 本地开发
npm run serve

# 生产构建
npm run build
```
